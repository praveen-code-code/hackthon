Rules & Guidelines
Read carefully before you start building

This hackathon is organised by Solete. The goal is to build a real, working backend API with a connected frontend for one of the 10 project tracks above. You are free to use any AI tool — ChatGPT, Claude, Gemini, Copilot, or anything else — to help write, debug, and plan your code. What matters is that you understand what you've built.

1️⃣
Pick One Project
Choose any one track from the 10 options. Each participant or team works on a single project. No duplication within a team.
2️⃣
Build the Backend API
Use Flask or FastAPI with PostgreSQL. Implement all listed endpoints with JWT authentication and proper error handling.
3️⃣
Design a Frontend
Create a working UI (HTML/CSS/JS or React) that connects to your API and demonstrates all core features.
4️⃣
Follow the DB Schema
Use the exact table and column names provided. You may add extra columns but all required fields must be present.
5️⃣
AI Tools Are Allowed
ChatGPT, Claude, Gemini — all fine. But during evaluation you must explain your code. If you can't, marks will be deducted.
7️⃣
No Plagiarism
AI-generated code is fine. Copy-pasting another participant's code is not allowed and will result in disqualification.
🌟
Bonus Points
Extra credit for live deployment (Render / Railway), a Postman collection, unit tests, and polished UI design.
